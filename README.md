
# Printers Manager

Aplication for managment a list of printers.




## Tech Stack

**Client:** React, Redux, Axios

**Server:** Node, Express, Mongoose, Jwtoken


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
